# Project Brief

## Overview
Marmara_Not is a Django-based web application designed to serve as a platform for sharing and accessing university lecture notes. It targets students of Marmara University (implied by the name) to facilitate academic resource sharing.

## Core Goals
- **Centralized Note Sharing**: Provide a platform where students can upload and find lecture notes.
- **Organized Structure**: Categorize notes by Faculty, Department, and Course.
- **Quality Control**: Implement an approval system for uploaded notes.
- **User Management**: Allow users to register, upload notes, and track their submissions.

## Key Features
- **Hierarchical Navigation**: Faculties -> Departments -> Courses -> Notes.
- **Note Upload**: Registered users can upload notes (files) with descriptions.
- **Approval Workflow**: Admin approval required for notes to be visible.
- **Search/Filter**: (Implicit) Easy access to notes via the hierarchical structure.
